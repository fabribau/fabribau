Sistema integral para la gestión académica universitaria, diseñado para facilitar el trabajo de docentes y administrativos.

## El problema

Las universidades suelen depender de planillas de Excel para gestionar notas, asistencia y seguimiento de alumnos. Esto genera:

- Duplicación de datos
- Errores humanos frecuentes
- Dificultad para generar reportes

## La solución

Desarrollé una plataforma web que centraliza toda la gestión académica:

- **Dashboard del docente**: vista general de cursos, alumnos y calificaciones
- **Registro de asistencia**: con posibilidad de QR o registro manual
- **Sistema de calificaciones**: parciales, trabajos prácticos y exámenes finales
- **Reportes automáticos**: generación de actas y estadísticas por curso

## Aprendizajes

Este proyecto me enseñó mucho sobre diseño de bases de datos relacionales y la importancia de pensar la UX para usuarios no técnicos.
