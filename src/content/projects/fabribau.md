Mi sitio web personal, donde publico artículos técnicos, muestro mis proyectos y comparto lo que aprendo.

## Características

- **Blog con MDX**: escribo posts con Markdown extendido que soporta componentes interactivos
- **Portafolio de proyectos**: cada proyecto tiene su propia página con descripción, stack y links
- **Tema oscuro/claro**: detección automática con toggle manual
- **SSG por defecto**: rendimiento máximo, solo SSR donde es necesario
- **SEO optimizado**: meta tags, Open Graph, JSON-LD structured data

## Stack técnico

El sitio está construido con Astro en modo estático, con endpoints SSR solo para las funciones que lo requieren (likes, analytics, newsletter).

Los estilos usan Tailwind CSS v4 con CSS custom properties para el sistema de temas.
